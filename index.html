HTML Cheatsheet

HTML table tags
Tag Description
<table> This is the main container for creating a table. It defines the beginning and end of a table.
<tr> This stands for "table row". It defines a row in the table and contains one or more cells.
<th> This stands for "table header". It is used to define a header cell in the table. The text in a header cell is usually bold and centered by default.
<td> This stands for "table data". It defines a regular cell in the table and contains the actual data to be displayed.
<thead> This tag is used to group the header content in a <table>
<tbody> This tag is used to group the body content in a <table>
<tfoot> This tag is used to group the footer content in a <table>
<caption> This tag is used to add a caption for the table, which is displayed above or below the table.
colspan and rowspan attributes These attributes are used to specify how many columns or rows a table cell should span, respectively. It allows you to merge cells horizontally or vertically.
<colgroup> This element allows you to group one or more columns together and apply styles to them as a group.
<col> This element is used inside <colgroup> and it represents a column within a table.
HTML table attribute
Attribute description
border Specifies the width of the border around the table
cellspacing Specifies the space between cells in the table
cellpadding Specifies the space between the cell content and the cell border
width Specifies the width of the table
height Specifies the height of the table
align Specifies the horizontal alignment of the table
valign Specifies the vertical alignment of the table
caption Specifies a caption for the table
summary Provides a summary of the table's structure

HTML Form Tags
Tag Description
<form> The main container for creating a form. It defines the beginning and end of a form and it contains various controls such as text fields, checkboxes, and buttons.
<input> The most versatile form control and it is used to create various types of form controls, such as text fields, checkboxes, radio buttons, and buttons.
<label> This tag is used to provide a text description for a form control. It is associated with the control using the "for" attribute, which should match the "id" attribute of the control.
<select> This tag is used to create a drop-down list of options for the user to select from.
<option> This tag is used within a `<select>` element to define the options in the drop-down list.
<textarea> This tag is used to create a multi-line text input field. It contains the `cols` and `rows` attribute to specify the number of columns and rows.
<fieldset> This tag is used to group related form controls and apply styles to them as a group.
<legend> This tag is used to provide a caption for a fieldset element, which is displayed as a label for the group of controls.
<button> Creates a clickable button. It can be used to trigger JavaScript events.
<datalist> This tag is used to create a list of options that can be used with an <input> element as a pre-defined list of possible values for the input field.
<output> This tag is used to display the result of a calculation.
Form Attribute
Tag Description
action This attribute specifies the URL of the server-side script that will process the form data. It is used to indicate where the data should be sent when the form is submitted.
method This attribute specifies how the form data should be submitted to the server. The two most commonly used values for this attribute are "GET" and "POST". GET is used for sending data to the server in the form of query parameters, while POST is used for sending data in the message body of the request.
target This attribute is used to specify where to display the response that is received after submitting the form. For example, you can use the value "_blank" to open the response in a new window or tab.
name this attribute is used to specify a name for the form and can be used to reference the form in JavaScript code.
Form Input Attribute
Tag Description
text Creates a single-line text field. This is the default input type.
password Creates a single-line text field, but the input is obscured, so that the entered text is not visible on the screen.
checkbox Creates a checkbox control. The user can select or deselect the checkbox to indicate a true or false value.
radio Creates a radio button. The user can select one option from a group of related options.
submit Creates a submit button. When clicked, the form data is sent to the server for processing.
reset Creates a reset button. When clicked, the form data is cleared.
image Creates an image as a submit button. When clicked, the form data is sent to the server for processing.
button Creates a clickable button. It can be used to trigger JavaScript events.
hidden Creates a hidden field. The value of the field is not visible to the user and is sent to the server with the form data.
file Creates a file picker that allows the user to select a file to be uploaded to the server.
number Creates a text field that accepts numeric input only
range Creates a slider control that allows the user to select a value within a given range.
color Creates a color picker that allows the user to select a color.
date Creates a date picker that allows the user to select a date.
email Creates a text field that accepts email address and check for the email pattern.
tel Creates a text field that accepts telephone number.
url Creates a text field that accepts URL and check for the url pattern.
